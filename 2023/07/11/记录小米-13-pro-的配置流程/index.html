记录小米 13 Pro 的配置流程

发表于 2023-07-11 更新于 2023-07-12 分类于 杂项 Valine：

实在忍受不了 Apple 的 iPhone 12 狗屎玩意了，趁 618 换成了 12GB + 512GB 的小米 13 Pro。虽然安卓的配置确实要比 iOS 复杂不少，无法做到开箱即用，但配置好了之后的体验可太舒服了。也因此在这里对整个配置流程做一个详尽的记录，毕竟这些配置又细又复杂，人脑子确实记不住。

先介绍一下整体流程，一共分两个部分：

第一部分为 rom 层面，共八步：

  1. 解锁手机
  2. 刷入 xiaomi.eu rom
  3. 刷入 magisk-delta 获得 root 权限
  4. 启用 magisk-delta 中的 Zygisk 和 Zygisk-LSPosed
  5. 安装适配 xiaomi.eu rom 的本地化模块和修复 FCM 推送的 fcmfix
  6. 打开 magisk-delta 中的 magiskhide 并强制使用超级用户列表
  7. 隐藏 magisk-delta 和 LSPosed
  8. 将官方的恢复模式替换为 twrp

第二部分为程序层面，共四步：

  1. Google Play 商店内进行软件安装
  2. 安装小米商城后，进行软件安装
  3. 系统设置
  4. 各个软件的内部设置

    rom 层面

解锁手机

  • 因为不解锁手机是无法刷 rom 的，所以这一步是前提。

  • 如果手机已在使用中，则必须先备份数据，因为解锁时会清除所有手机数据。

  • 解锁工具 miflash unlock 只有 Windows 端，所以并不能在 Linux 中进行解锁。在小米官网下载并更新到最新版本的 miflash unlock，目前 2023 年 6 月底的最新版本为 7.6.602.42。

  • 在手机开机或重启的开机阶段，保持按住音量下键，进入 fastboot 模式。

  • 将 fastboot 模式下的手机连接到 Windows 主机，打开 miflash unlock。

  • 对于从未解锁过的手机，并不能直接解锁，需要此时在 miflash unlock 中申请解锁并等待 168 小时才能正式解锁。

  • 对于已经解锁过或者等待 168 小时之后的手机，此时直接在 miflash unlock 解锁手机即可。

  • 解锁完成后，手机会自动重启。

刷入 xiaomi.eu rom

  • 选择的是非官方的 xiaomi.eu rom，由 xiaomi.eu 维护。这个版本的 rom 自大陆的原始 rom 修改而来，去除了广告和为大陆服务的本地化内容（例如手机黄页、小爱同学等），同时引入了原生的 Google 框架。

  • 根据手机型号，下载开发版的 weekly roms 或稳定版 stable roms。

  • 解压下载的 rom 压缩包后，将手机以 fastboot 模式连接电脑，执行解压内的对应 bat。

    • windows_fastboot_first_install_with_data_format.bat：新安装 rom 或者降级时用，会抹除手机上的所有数据。

    • windows_fastboot_update_rom：rom 更新时用，不会抹除数据。

  • 选择新安装 rom 时会有以下提示，输入 Y 安装即可。安装完成后手机会自动重启。

刷入 magisk-delta 获得 root 权限

  • 由于 xiaomi.eu rom 去除了为大陆服务的本地化程序，而这对于大陆用户而言是相当不便的，唯一的补救措施是安装社群管理的本地化工具箱，而本地化工具箱要求手机安装了 magisk 和 LSPosed，并取得手机的 root 权限。

  • 使用 magisk-delta 而非原生 magisk 的原因是 delta 版本拥有更好的防 root 检测功能。原生的 magisk 即使安装了防 root 检测模块，仍然有部分程序（例如数字人民币、Barclays 等银行类程序）能检测出手机已经被 root 从而无法启动。

  • 开机，执行 Google 账号登录和对应设定。连接 Google 的通常做法是打开 Windows 主机的 v2rayn 内的“允许局域网连接”功能，随后将手机连接到 Windows 主机所在的无线网络，选择无线网络选项内的“代理”，调整为手动模式并填入 Windows 主机的本地 ip 和使用的 http/https 或 socks 端口。Windows 主机的本地 ip 一般可以在路由器中查看。

  • 将手机直接连接 Windows 主机，USB 模式允许文件传输。将 magisk-delta 的 apk 安装文件和 xiaomi.eu rom 的 images 文件夹内的 init_boot.img 传入手机。

  • 在手机内打开 apk 文件安装 magisk-delta 后，打开 magisk-delta 的界面。此时界面内 magisk 的“当前”状态为无法获取。

  • 点击 界面内 magisk 右侧的安装按钮，点击“选择并修补一个文件”选项，选中传入的 init_boot.rom。

  • magisk-delta 会生成一个 magisk_patched 前缀的 img 文件，将这个文件从手机中取出放入 Windows 主机内。

  • 重启手机并进入 fastboot 模式。在 Windows 主机中使用 scoop install adb 安装 adb 工具后，在终端内执行 fastboot 命令如下，将 magisk-delta 生成的 img 文件刷入手机。

启用 magisk-delta 中的 Zygisk 和 Zygisk-LSPosed

  • 这一步的目的是启用 magisk-delta 内的 Zygisk 和 Zygisk 下的 LSPosed，这两个工具都是使用本地化模块的必要前提。

  • 打开 magisk-delta，此时会显示“需要修复运行环境”。点击继续，完成后会自动重启手机。

  • 再次打开 magisk-delta，界面内的 magisk 的“当前”会显示安装的 magisk-delta 版本，目前 2023.07 为 fdac22ba0delta (25210)。

  • 主页右上角齿轮设置中启用 Zygisk 和 Bootloop 保护，暂时不要打开 magiskhide。

  • 在 magisk-delta 的模块标签页中选中 Zygisk-LSPosed 的 zip 安装包进行安装，安装完成后重启手机。

安装适配 xiaomi.eu rom 的本地化模块和修复 FCM 推送的 fcmfix

  • 通过在 magisk-delta 中安装本地化模块和在 LSPosed 中启用本地化工具箱来完成对 xiaomi.eu rom 中去除的本地化部分的还原。

  • 对于 xiaomi.eu rom 而言 FCM 消息推送是完全正常的，而在安装了本地化模块后，FCM 消息推送会出现和大陆版原 rom 一样的 FCM 无法推送已经停止的应用的消息这个问题，在 FCM diagnostic 内可以看到类似 fcm failed to broadcast to stopped app 的报错信息。因此需要安装 fcmfix 并在 LSPosed 启用来修复这个问题。

  • 下载 MIUI EU 欧洲版 本地化 Magisk 模块，解压出其中的 MiuiEuLocalization.ini 文件，打开并根据想要恢复的应用进行所需的修改。

  • 下载 fcmfix。

  • 将手机连接至 Windows 主机，将本地化模块压缩包、ini 文件和 fcmfix 的安装文件一并放入手机的 Download 文件夹内。在 magisk-delta 中的模块页面安装本地化模块，而 fcmfix 直接打开安装即可。

打开 magisk-delta 中的 magiskhide 并强制使用超级用户列表

  • 这一步的目的相当于开启 magisk-delta，或者说 root 权限的白名单。通过这个方法开启的白名单是目前效果最好的。

  • 进入 magisk-delta，主页右上角齿轮设置中启用 magiskhide 并打开“强制使用超级用户列表”，重启手机。

  • 此时手机开机会发现本地化模块所安装的天气等一系列应用程序全部都会报错，包括桌面上的应用程序也无法正确显示名称。这正是由于已经打开了 root 权限的白名单，而这些需要 root 权限的程序均不在白名单内。

  • 为了保险起见，此时进入 magisk-delta 并在主页右上齿轮设置中选择“配置 magiskhide”，在配置页面的右上角三点内勾选“显示系统应用”和“显示系统程序”，然后为当前系统内安装的所有程序的所有部分全部授予 root 权限。虽然比较麻烦，但这是最保险的做法，而且这个时间点也是最合适的，没有任何其他另外安装程序的干扰，只有系统必须的程序，从而不会出现误给 root 权限的情况。

  • 重启手机，本地化模块即可以重新恢复正常。打开 LSPosed，此时会提示创建快捷方式，暂时先点击确认，随后在模块页面中启用本地化工具箱和刚安装的 fcmfix。

隐藏 magisk-delta 和 LSPosed

  • 这一步是隐藏手机 root 的最后一步。

  • 打开 magisk-delta，主页右上齿轮设置中选择“隐藏 magisk 应用”，此时会弹出安装未知应用的权限确认页面，在列表内中找到 magisk-delta 并“允许来自此来源的应用”。安装后打开隐藏后的 magisk-delta，提示是否需要添加原始的 magisk-delta 的快捷方式到桌面，此处选择取消即可。

  • 打开 LSPosed，在设置页面中创建打开寄生管理器的快捷方式，随后关闭状态通知，即完成了 LSPosed 的隐藏。
